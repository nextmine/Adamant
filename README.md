# Adamant

Adamant - это высокопроизводительное и гибко настраиваемое ядро сервера для Minecraft Bedrock Edition.

## Особенности

- Поддержка последних версий Minecraft Bedrock Edition
- Высокая производительность и стабильность
- Гибкая система конфигурации
- Простота использования и настройки
- Открытый исходный код

## Установка и запуск

1. Клонировать репозиторий:
```
git clone https://github.com/yourusername/Adamant.git
```
2. Перейти в каталог проекта:
```
cd Adamant
```
3. Собрать и запустить сервер:
```
go build
./Adamant
```
